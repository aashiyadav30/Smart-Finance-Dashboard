# Smart Finance Dashboard

A minimal and professional finance dashboard built using modern web technologies.  
Designed with a clean, government-style UI to present financial data in a structured and user-friendly way.

## Live Demo

https://smart-finance-dashboard-two.vercel.app/

## Features

- Clean and minimal dashboard interface
- Government-style UI (simple, structured, non-flashy)
- Responsive layout
- Financial data visualization
- Component-based architecture
- Fast and optimized build using Vite
- Easy deployment with Vercel

## Tech Stack

- React.js
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/aashiyadav30/Smart-Finance-Dashboard.git
cd Smart-Finance-Dashboard
2. Install dependencies
npm install
3. Run the development server
npm run dev
4. Build for production
npm run build
Deployment

This project is deployed on Vercel.

To deploy your own version:

Push your code to GitHub
Import the repository into Vercel
Click deploy
